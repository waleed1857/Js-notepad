About Notepad

This notepad has three sections.

  • New Note Section
  • Saved Notes Section
  • Delete Notes Section

On New Note section, user will be able to make notes. This section has an input field to enter the key. This key specify all the user's notes. This section has 3 button, one button is to save note, one is to retrieve note, while the other one is to update note. These 3 functionality works with the key which the user will input. Note can be updated by pressing update button after retrieving data. It also has a side panel to go to the other sections.

In Saved Section, user will be able to see all the saved notes with key written on top.

In Delete Section, user will be able to delete the specific note by input the key of that note in the field.
